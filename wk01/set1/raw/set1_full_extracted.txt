ISA vs. Microarchitecture
ยง ISA: Specification of a set of definite instructions that the computer
can carry out
ยง All computers (CPUs/microprocessors) perform the same set of
basic instructions
ยง ADD, MULTIPLY, DIVIDE, MOVE, BRANCH
ยง Two manufacturers might differ in which set of basic instructions
ยง Microarchitecture: Implementation of the ISA using circuits
